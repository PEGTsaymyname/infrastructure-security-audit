# Аудит безопасности и анализ защищенности ИТ-инфраструктуры предприятия

Выпускной практический проект, выполненный в рамках программы профессиональной переподготовки «Информационная безопасность».

## 📝 Краткое описание проекта
Проект посвящен комплексному тестированию на защищенность (Penetration Testing) смоделированной корпоративной сети предприятия. В ходе работы был проведен сбор информации о периметре, сканирование активных сервисов, выявление критических уязвимостей конфигурации и моделирование векторов потенциальных атак.

---

## 🛡️ Использованный технический стек и инструменты
* **OS:** Kali Linux, Ubuntu Server
* **Сетевой анализ и сканирование:** Nmap, Wireshark, Netdiscover
* **Анализ веб-приложений:** Burp Suite (OWASP Top 10)
* **Аудит парольной политики:** Hydra (brute-force симуляция)
* **Поиск уязвимостей:** Поиск публичных эксплойтов (Exploit-DB, Searchsploit)

---

## 📈 Ключевые результаты исследования
1. Выявлены критические уязвимости конфигурации веб-сервера и слабые парольные политики удаленного доступа (служба SSH).
2. Успешно смоделирована и задокументирована цепочка эксплуатации уязвимостей (Kill Chain).
3. Разработан детализированный план митигации рисков (Remediation) и даны рекомендации для системных администраторов по укреплению безопасности (Hardening).
